DESCRIPTION
==================================================
```steinhardt``` library.


GETTING THE CODE
==================================================
* To get the code:
```
git clone git@github.com:pgniewko/steinhardt.git
```

* To obtain the most recent version of the code:
```
git pull origin master
```

COMPILING AND INSTALLATION - LINUX/UNIX
==================================================

* Build the library:
```
make build
```

* Compile and install library:
```
make install
```
